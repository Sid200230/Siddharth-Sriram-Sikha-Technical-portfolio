The project aimed to address limitations in traditional masks by creating a design that protected both the user and their surroundings. Leveraging advanced mechanical design tools and simulation techniques, we conceptualized, prototyped, and optimized this device for maximum efficiency and usability.

Using Autodesk Fusion 360, we designed a modular face shield with an integrated Tesla valve to regulate airflow, ensuring effective filtration. Our solution included a multi-layer filtration system: ethanol-coated PLA, a Tesla valve, and synthetic microfiber. This design achieved improved air filtration while maintaining user comfort through proper ventilation. Post-design, the product was rigorously tested on users across age groups, validating its functionality and effectiveness.

By collaborating on challenges such as airflow optimization, material durability, and manufacturability, we enhanced the face shield’s reliability. This project not only demonstrated my expertise in CAD modeling, CFD simulations, and additive manufacturing but also highlighted my ability to innovate under real-world constraints. The final product addressed critical health and safety needs while aligning with sustainable engineering practices, showcasing a blend of technical precision and user-focused design.

Evaluation of Vibration Techniques to Measure Scuffing in Packaged Products

Overview: This project aimed to address scuffing in packaging, a critical issue impacting product aesthetics and durability during transportation. By designing an innovative scuffing rig, performing experimental vibration analysis, and deriving actionable insights, the project provided a comprehensive solution to improve packaging resilience. Design and Development: The project began with the conceptualization of a scuffing rig to simulate real-world vibrations experienced during transport. Using SolidWorks, I designed two iterations of the rig, refining its functionality after identifying friction and motion issues in the initial prototype. To improve stability and accuracy, I incorporated springs and bushings sourced from McMaster-Carr in the second iteration, which eliminated wobbling and ensured smooth vertical motion. The prototype was fabricated using Ultimaker 5S 3D printers and optimized with Cura software, showcasing advanced manufacturing techniques and material efficiency.

Experimental Analysis: With the rig complete, I conducted vibration testing using a Lansmont Saver 9X30 data logger to replicate real-world conditions. By analyzing vibration frequencies between 0-300 Hz, I identified the critical range of 3-40 Hz for optimal scuffing tests, significantly reducing testing time without sacrificing accuracy. I processed the test data using MATLAB, employing Power Spectral Density (PSD) and Fast Fourier Transform (FFT) to quantify scuffing percentage through image degradation analysis.

Results and Insights: The testing revealed key vibration parameters contributing to scuffing, leading to recommendations for optimized packaging designs. By narrowing the testing frequency and refining material choices, the project provided a scalable approach for reducing damage during transportation. This work not only highlighted my expertise in design, analysis, and simulation but also underscored my ability to integrate advanced tools and techniques to solve practical engineering challenges.

Implementation of Reaction-Diffusion Textures for Haptic Experiments

This project explored the innovative conversion of reaction-diffusion patterns into 3D printed textured boards for use in haptic experiments. By combining advanced computational modeling, CAD design, and 3D printing, the project bridged the gap between digital pattern generation and tangible textures, achieving significant progress in tactile research.

The core of the project lay in developing two distinct methodologies for converting intricate 2D diffusion patterns into 3D textures:

1) Python-Based Contour Detection: Using the Gray-Scott model, patterns were generated and refined with scikit-image’s contour detection algorithm and the Ramer–Douglas-Peucker algorithm for smoother curves. The contours were transformed into 3D models using Fusion 360 Automate API, which included edge filleting and the addition of a base to enhance structural integrity. The final outputs were exported as STL files for 3D printing.

2) MATLAB-Based Mesh Generation: This method utilized MATLAB’s meshz and Delaunay triangulation techniques to create 3D models from the diffusion patterns. The STL files generated were further refined in Fusion 360, ensuring accurate surface representation and base integration.

The 3D models were printed using a resin printer, followed by meticulous polishing with grit sandpaper and resin spraying to enhance transparency and tactile clarity. The textured boards, fabricated at a cost-effective $15 each, met the requirements for haptic experiments: visual transparency, modifiable textures, and scalability for various setups. This project demonstrated proficiency in Python, MATLAB, Fusion 360, and 3D printing, along with expertise in reaction-diffusion modeling and surface engineering. The successful implementation highlights the capability to integrate computational design with physical manufacturing, paving the way for advancements in tactile research and experimental design.

End to End Data Engineering Process on the NSL- KDD dataset

In this project, the focus was on developing a system capable of predicting and classifying network intrusions using the NSL-KDD dataset, which comprises around 125,000 data points. The aim was to analyze the dataset, identify patterns, and similarities between intrusions to enhance network security.

The project began with extensive exploratory data analysis (EDA) and data cleaning to ensure data quality and reliability. Various data preprocessing techniques were employed to prepare the dataset for machine learning model implementation. The use of PySpark facilitated efficient data processing, while PostgreSQL aided in managing the dataset. Scikit-learn was utilized for implementing machine learning models, and Google Cloud Platform (GCP) tools were leveraged for model training and deployment.

After performing EDA, data cleaning, and implementing machine learning models, the system demonstrated robustness, accuracy, and scalability in predicting and classifying network intrusions. The combination of PySpark, PostgreSQL, Scikit-learn, and GCP tools played a crucial role in achieving efficient data processing, model training, and deployment.

The project's successful outcome underscores the importance of thorough data preprocessing, effective machine learning model implementation, and utilizing cloud-based platforms like GCP for scalability and deployment. The system's capabilities in network intrusion detection showcase its potential as a comprehensive and effective solution for enhancing cybersecurity measures.
